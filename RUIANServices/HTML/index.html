Tento portál umožňuje využívat kopii databáze Registru Územních Identifikací, Adres a Nemovitostí (RÚIAN) pomocí webových služeb.

Jednotlivé služby je možné využívat pomocí standardů Representational State Transfer (REST) a Simple Object Access Protocol (SOAP)/Web Services Description Language (WSDL).
